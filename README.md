# 📖ThePower Dashboard & Data Analysis
En este proyecto he aplicado los conocimientos aprendidos del módulo “Dashboard & Análisis de Datos”. Para realizar este proyecto se han seguido las especificaciones recogidas en la lección correspondiente del módulo.
🎯 Objetivo del proyecto
Realizar un análisis exploratorio de un conjunto de datos de elección propia, con las siguientes condiciones:
-	Mínimo 10 columnas 
-	Al menos 2000 filas 
-	Creación de un dashboard interactivo 
-	Elaboración de un informe explicativo del análisis
  
El objetivo es transformar los datos en información útil para la toma de decisiones mediante visualizaciones y análisis descriptivo.

## 📌 Requisitos del proyecto
```
Requisito	              Estado	          Detalles
Mínimo 10 columnas        ✅ Cumplido    	  11 columnas analizadas
Al menos 2,000 filas	  ✅ Cumplido	      11,971 transacciones (post-limpieza)
Limpieza de datos	      ✅ Cumplido         604 registros eliminados, 9,220 imputados
Análisis descriptivo	  ✅ Cumplido         Estadísticas completas + hallazgos
Dashboard interactivo	  ✅ Cumplido         Segmentadores, gráficos dinámicos
Informe explicativo       ✅ Cumplido	       Metodología + recomendaciones priorizadas
```


## 🛠 Herramientas utilizadas
Microsoft Excel:
-	Power Query (limpieza y transformación) 
-	Tablas dinámicas 
-	Gráficos dinámicos 
-	Segmentadores (slicers)

## 📊 Resultados y conclusiones
Durante el desarrollo de este proyecto se realizó un proceso completo de análisis de datos en Excel, desde la limpieza inicial hasta la creación de visualizaciones interactivas en un dashboard.
El trabajo permitió:
-	Detectar y corregir valores nulos, duplicados y errores de formato 
-	Estandarizar variables para facilitar el análisis 
-	Explorar patrones relevantes mediante tablas dinámicas
-	Visualizar tendencias y relaciones entre variables clave 
-	Construir un dashboard interactivo para la toma de decisiones 
Este proyecto ha permitido reforzar habilidades clave en análisis de datos, especialmente en la transformación, exploración y visualización de información.


## 🎯 Principales aprendizajes
-	Limpieza y preparación de datos en Excel (Power Query)
